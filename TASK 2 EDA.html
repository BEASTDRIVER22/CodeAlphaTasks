Titanic Dataset - Exploratory Data Analysis (EDA)¶

Questions for Analysis¶

  • Does gender affect survival rate?
  • Does passenger class impact survival?
  • Does age influence survival chances?
  • Does fare (ticket price) relate to survival?
  • Do family members (SibSp, Parch) affect survival?

Hypotheses¶

  • Females are more likely to survive than males
  • Higher class passengers have better survival chances
  • Younger passengers may survive more
  • Higher fare increases survival probability

Data Cleaning¶

  • Cabin column dropped due to excessive missing values
  • Missing values in Age column filled using median

Data Issues Identified¶

  • Missing values in Age and Cabin
  • Cabin column is not useful due to excessive missing data
  • Dataset may require encoding for machine learning

Conclusion¶

  • Gender and passenger class strongly influence survival
  • Higher fare passengers had better survival chances
  • Age was not a major factor
  • Data cleaning improved dataset usability
  • Hypotheses were mostly supported by analysis
